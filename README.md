#### 安装教程

1.  克隆仓库 
```
  git clone https://github.com/liqingyang-1998/YOHO-BUY.git
```
2.  进入项目 
```
cd YOHO-BUY
```
3.  加载npm包 
```
npm i 或者 cnpm i 或者 yarn
```

#### 使用说明
1. 运行项目 
```
npm run dev 或者 yarn dev
```
